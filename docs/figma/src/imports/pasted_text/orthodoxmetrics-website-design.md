PROJECT:
OrthodoxMetrics Website Redesign

You are redesigning the public-facing frontend pages for a SaaS platform called OrthodoxMetrics.

OrthodoxMetrics is a records management and analytics platform designed for Orthodox Christian churches and parishes. The platform digitizes historical sacramental records (baptisms, marriages, funerals), organizes them into structured databases, and allows churches to search, manage, and analyze their historical data.

The visual tone must reflect:

• Orthodox Christian heritage
• professionalism
• trust
• historical preservation
• modern SaaS capability

This is NOT a typical startup website. The tone should feel:
traditional, respectful, authoritative, and long-term stable.

------------------------------------------------------------

REFERENCE DESIGN

Use the redesigned **About Us page** as the PRIMARY design language reference.

Adopt its:

• typography
• color palette
• spacing system
• section layout rhythm
• card styling
• icon style
• background treatments
• visual hierarchy

The About page establishes the base **design system** for the rest of the site.

------------------------------------------------------------

DESIGN SYSTEM REQUIREMENTS

Maintain a consistent design system across all pages.

Typography:

• Serif headline font for titles
• Clean sans-serif for body text
• Strong hierarchy between:
  - hero titles
  - section headers
  - body copy
  - supporting text

Color system should include:

Primary:
Deep Orthodox purple

Accent:
Gold / warm accent

Neutral:
Dark charcoal
Light backgrounds
Soft grays for containers

Spacing should follow a clear scale so that sections breathe and feel premium.
