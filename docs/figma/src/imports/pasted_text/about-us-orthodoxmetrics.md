Prompt — Redesign "About Us" Page for OrthodoxMetrics

Design a professional, modern "About Us" page for a software platform called Orthodox Metrics, which provides digital record management for Orthodox Christian parishes (baptisms, marriages, funerals, parish records).

The design must feel:

• professional
• calm and trustworthy
• respectful of Orthodox tradition
• modern SaaS quality

Avoid generic startup aesthetics.
